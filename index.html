Map Visualisation - Critically Polluted Industrial Area/Clusters based on CEPI Index.

What is the map trying to represent?
=> This is a map showing clusters of Critically Polluted Industrial Clusters/Areas in India.

What does the blue markers represent?
=> The blue markers represent the Industrial Clusters/Area.

How can I view the CEPI index Scores?
=> You can click on the markers () to get additional Information about the CEPI Index Scores.

Additional Information
=> Each Industrial Cluster has a CEPI index value comprising of 3 dimensions.The index captures the dimensions of environment including Air, Water and Land.

What is CEPI Index ?
=> Comprehensive Environmental Pollution Index (CEPI) comprises of the scores of the critically polluted industrial clusters/areas. The index captures the various dimensions of environment including air, water and land.

How can I view the CEPI index Scores?
=> You can click on the markers () to get additional Information about the CEPI Index Scores.
